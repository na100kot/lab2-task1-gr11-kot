# Лабораторная работа №2

Данный репозиторий содержит результаты выполнения лабораторной работы №2 по теме "Автоматизация разработки консольных приложений". Проект демонстрирует навыки работы с Git, подмодулями, утилитой make и автоматизацией через GitHub Actions.

## Modules

В проекте реализованы следующие модули (подмодули):

* **[tpmp-lab2-task3](https://github.com/na100kot/tpmp-lab2-task3)** — Работа со структурами данных. Реализована модель КИС (Клиент-Интерфейс-Сервер). Программа позволяет вводить данные о людях (структура NOTE1), сортировать их по алфавиту и выполнять поиск по месяцу рождения.
* **[tpmp-lab2-task4](https://github.com/na100kot/tpmp-lab2-task4)** — Работа с текстовыми файлами. Реализовано чтение базы данных рабочих из файла, фильтрация сотрудников, поступивших на работу в 2020 году, и сохранение результата в новый текстовый файл.
